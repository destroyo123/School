# AME 323 - Gas Dynamics, Project 1: Solving the Taylor-Maccoll equations
## About
  This project is part of the University of Arizona's gas dynamics (AME323) class, and involves using MATLAB, Python, or another language of choice to solve the Taylor-Maccoll equations for conical shocks & calculate the shock angle (β) for a given Mach number (M), and the cone semivertex angle (θ).
