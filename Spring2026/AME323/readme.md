# AME323 Folder: Gas Dynamics
## About
Welcome to my folder of code-related materials I used and/or created as part of the gas dynamics (AME323) class, taught by Dr. Stuart Alex Craig.

### mathFunctions
These are a bunch of gas-dynamics-related functions I wrote in MATLAB, separated as individual files, but in one folder. The idea behind this is that you can then add this folder to your matlab PATH (or I could automate it for anyone that clones this Github repo) and can easily access the functions without having to type, calculate, or program them by hand. Most of them are sourced from NACA1135, or from AME323 lectures notes, or other sources on the internet. I didn't specify which come from where, as I wrote the code for everything except one function, which I cite.

### calculatorPrograms
A bunch of calculator programs for the Ti-84 plus CE python edition.
This has some Python programs to do analytical calculations of things that you'd otherwise need to find through flow tables or charts, giving you a more exact value than you'd get from the tables, and doing it faster than you would if you had to type it out yourself manually and do all the bookkeeping.

### Discussion306
This is the code I wrote for a discussion involving a diamond-shaped wedge airfoil in supersonic flow, generating lift and drag. It wasn't perfect due to some sign errors, and as of 3-23-26 I don't know if I fixed it yet. But there's a lot of work in it, and it's actually where I first compiled all the other math functions I wrote from other scripts.
